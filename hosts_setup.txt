# Configuración del archivo hosts para comunidades_v3.test
# 
# Para configurar el dominio local, sigue estos pasos:
#
# 1. Abre el Bloc de notas como Administrador
# 2. Abre el archivo: C:\Windows\System32\drivers\etc\hosts
# 3. Agrega esta línea al final del archivo:
#
# 127.0.0.1 comunidades_v3.test
#
# 4. Guarda el archivo
# 5. Reinicia el navegador
#
# Ahora podrás acceder a: http://comunidades_v3.test/
#
# Nota: Si tienes problemas, también puedes usar:
# 127.0.0.1 www.comunidades_v3.test
